Selecting Elements

There are multiple ways to select and group elements
You can select multiple items by holding Ctrl/Cmd or Shift and clicking the items
Same thing can be achieved in the Tree View
The only difference is that in the Tree View pressing Shift + Click allows selecting all siblings between two elements
You can also mouse-select by dragging over the desired items and holding Ctrl/Cmd pressed
If you are not dragging over a container you don't need to press Cmd/Ctrl
If you want to become a pro go to the Keyboard Shortcuts - Control - Selection and see all the ways you can select elements
Open the Keyboard Shortcuts panel by pressing Ctrl + Shift + /
